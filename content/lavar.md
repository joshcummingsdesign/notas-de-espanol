# Lavar

"Lavar" means "to wash." It's a regular -ar verb.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | lavo        |
| tú                  | lavas       |
| él/ella/usted       | lava        |
| nosotros            | lavamos     |
| ellos/ellas/ustedes | lavan       |

## Common Uses

| Spanish                      | English                      |
| ---------------------------- | ---------------------------- |
| Lavo los platos.             | I wash the dishes.           |
| ¿Lavas la ropa hoy?          | Are you washing clothes today? |
| Ella lava el carro.          | She washes the car.          |
| Me lavo las manos.           | I wash my hands.             |
