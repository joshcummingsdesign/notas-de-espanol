# Demonstratives

This, that, these, and those.

Spanish demonstratives agree in gender and number with the noun they modify.

## This / These

| Spanish | English | Example                    |
| ------- | ------- | -------------------------- |
| este    | this    | este libro (this book)     |
| esta    | this    | esta casa (this house)     |
| esto    | this    | ¿Qué es esto? (What is this?) |
| estos   | these   | estos libros (these books) |
| estas   | these   | estas casas (these houses) |

## That / Those

| Spanish | English | Example                    |
| ------- | ------- | -------------------------- |
| ese     | that    | ese libro (that book)      |
| esa     | that    | esa casa (that house)      |
| eso     | that    | ¿Qué es eso? (What is that?) |
| esos    | those   | esos libros (those books)  |
| esas    | those   | esas casas (those houses)  |

## That / Those (over there)

Used to stress a long distance or time.

| Spanish  | English | Example                              |
| -------- | ------- | ------------------------------------ |
| aquel    | that    | aquel avión (that plane)<br>Aquel viaje a México fue muy divertido. (stressing time) |
| aquella  | that    | aquella montaña (that mountain) |
| aquellos | those   | aquellos edificios (those buildings) |
| aquellas | those   | aquellas flores (those flowers) |
