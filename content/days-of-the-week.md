# Days of the Week

In Spanish, days of the week are masculine and not capitalized.

| Spanish      | English   |
| ------------ | --------- |
| el lunes     | Monday    |
| el martes    | Tuesday   |
| el miércoles | Wednesday |
| el jueves    | Thursday  |
| el viernes   | Friday    |
| el sábado    | Saturday  |
| el domingo   | Sunday    |

## Plural

Use "los" for plural. Note: only sábado and domingo change form.

| Spanish       | English    |
| ------------- | ---------- |
| los lunes     | on Mondays |
| los sábados   | on Saturdays |
| los domingos  | on Sundays |

## Common Uses

| Spanish                     | English                      |
| --------------------------- | ---------------------------- |
| Hoy es lunes.               | Today is Monday.             |
| El lunes tengo clase.       | On Monday I have class.      |
| Trabajo los viernes.        | I work on Fridays.           |
| ¿Puedes trabajar el lunes?  | Can you work on Monday?      |
| El lunes es mi día favorito. | Monday is my favorite day.  |
