# Aprender

"Aprender" means "to learn." It's a regular -er verb.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | aprendo     |
| tú                  | aprendes    |
| él/ella/usted       | aprende     |
| nosotros            | aprendemos  |
| ellos/ellas/ustedes | aprenden    |

## Common Uses

| Spanish              | English                |
| -------------------- | ---------------------- |
| Aprendo español.     | I'm learning Spanish.  |
| ¿Qué aprendes?       | What are you learning? |
| Ella aprende rápido. | She learns quickly.    |
| Aprendemos juntos.   | We learn together.     |

## Aprender + A + Infinitive

Use "aprender a" + infinitive to express learning how to do something:

| Spanish                      | English                          |
| ---------------------------- | -------------------------------- |
| Aprendo a cocinar.           | I'm learning to cook.            |
| Aprende a nadar.             | She's learning to swim.          |
| Aprendemos a hablar español. | We're learning to speak Spanish. |
