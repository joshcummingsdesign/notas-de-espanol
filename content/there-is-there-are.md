# Hay

"Hay" means "there is" or "there are." It comes from the verb "haber" and is used to indicate existence or availability.

## Examples

| Spanish                  | English                      |
| ------------------------ | ---------------------------- |
| ¿Hay café?               | Is there coffee?             |
| Hay mucha gente aquí.    | There are a lot of people here. |
| No hay problema.         | There's no problem.          |
| ¿Hay una farmacia cerca? | Is there a pharmacy nearby?  |
| Hay tres libros.         | There are three books.       |
| ¿Hay una mesa disponible adentro? | Is there a table available inside? |
| ¿Hay una mesa disponible afuera? | Is there a table available outside? |
