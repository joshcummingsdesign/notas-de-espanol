# Querer

"Querer" means "to want" or "to love." It's a stem-changing verb (e → ie).

## Present Tense (Presente)

Stem change: e → ie in all forms except nosotros.

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | quiero      |
| tú                  | quieres     |
| él/ella/usted       | quiere      |
| nosotros            | queremos    |
| ellos/ellas/ustedes | quieren     |

## Common Uses

### To Want

| Spanish             | English                 |
| ------------------- | ----------------------- |
| Quiero agua.        | I want water.           |
| ¿Qué quieres hacer? | What do you want to do? |

### To Love (people)

| Spanish              | English           |
| -------------------- | ----------------- |
| Te quiero.           | I love you.       |
| Quiero a mi familia. | I love my family. |

Note: "Te quiero" is commonly used for romantic love and close relationships. "Te amo" is more intense/passionate.

## Querer + Infinitive

| Spanish                    | English                   |
| -------------------------- | ------------------------- |
| Quiero comer.              | I want to eat.            |
| ¿Quieres bailar?           | Do you want to dance?     |
| Queremos aprender español. | We want to learn Spanish. |
