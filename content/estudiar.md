# Estudiar

"Estudiar" means "to study." It's a regular -ar verb.

## Present Tense (Presente)

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | estudio     |
| tú                  | estudias    |
| él/ella/usted       | estudia     |
| nosotros            | estudiamos  |
| ellos/ellas/ustedes | estudian    |

## Common Uses

| Spanish                      | English                  |
| ---------------------------- | ------------------------ |
| Estudio español.             | I study Spanish.         |
| ¿Qué estudias?               | What do you study?       |
| Ella estudia medicina.       | She studies medicine.    |
| Estudiamos en la biblioteca. | We study in the library. |
