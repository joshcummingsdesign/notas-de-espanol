# Poder

"Poder" means "to be able to" or "can." It's a stem-changing verb (o → ue).

## Present Tense (Presente)

Stem change: o → ue in all forms except nosotros.

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | puedo       |
| tú                  | puedes      |
| él/ella/usted       | puede       |
| nosotros            | podemos     |
| ellos/ellas/ustedes | pueden      |

## Common Uses

### Ability / Can

| Spanish                    | English                      |
| -------------------------- | ---------------------------- |
| Puedo hablar español.      | I can speak Spanish.         |
| ¿Puedes ayudarme?          | Can you help me?             |
| No podemos ir hoy.         | We can't go today.           |

### Permission

| Spanish                    | English                      |
| -------------------------- | ---------------------------- |
| ¿Puedo entrar?             | May I come in?               |
| ¿Puedo usar el baño?       | May I use the bathroom?      |
