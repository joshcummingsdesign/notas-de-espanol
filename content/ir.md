# Ir

"Ir" means "to go." It's a highly irregular verb.

## Present Indicative

Completely irregular conjugation.

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | voy         |
| tú                  | vas         |
| él/ella/usted       | va          |
| nosotros            | vamos       |
| ellos/ellas/ustedes | van         |

## Past Indicative

Completely irregular - shares the same conjugation as "ser."

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | fui         |
| tú                  | fuiste      |
| él/ella/usted       | fue         |
| nosotros            | fuimos      |
| ellos/ellas/ustedes | fueron      |

## Common Uses

| Spanish                  | English                    |
| ------------------------ | -------------------------- |
| Voy al supermercado.     | I'm going to the store.    |
| ¿Vas a la fiesta?        | Are you going to the party?|
| Vamos a la playa.        | We're going to the beach.  |

## Ir a + Infinitive

Use "ir a" to express the near future (going to):

| Spanish                  | English                    |
| ------------------------ | -------------------------- |
| Voy a comer.             | I'm going to eat.          |
| Vas a estudiar.          | You're going to study.     |
| Vamos a salir.           | We're going to leave.      |

## Ir + a + Location + a + Infinitive

Use this structure to say where you're going and why:

| Spanish                          | English                            |
| -------------------------------- | ---------------------------------- |
| Voy a la tienda a comprar manzanas. | I'm going to the store to buy apples. |
| Vas al gimnasio a entrenar.      | You're going to the gym to train.  |
| Vamos a la biblioteca a estudiar.| We're going to the library to study.|
