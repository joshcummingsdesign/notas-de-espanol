# Estar

"Estar" means "to be" for temporary states, conditions, locations, and emotions. It's an irregular verb.

## Present Indicative

Irregular in all forms.

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | estoy       |
| tú                  | estás       |
| él/ella/usted       | está        |
| nosotros            | estamos     |
| ellos/ellas/ustedes | están       |

## Common Uses

| Spanish                | English                |
| ---------------------- | ---------------------- |
| ¿Cómo estás?           | How are you?           |
| Estoy bien.            | I'm fine.              |
| Ella está cansada.     | She is tired.          |
| Estamos en la casa.    | We are at the house.   |
| ¿Dónde está el baño?   | Where is the bathroom? |
