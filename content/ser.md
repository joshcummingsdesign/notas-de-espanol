# Ser

"Ser" means "to be" for permanent characteristics, identity, origin, time, and occupation. It's an irregular verb.

## Present Indicative

Irregular in all forms.

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | soy         |
| tú                  | eres        |
| él/ella/usted       | es          |
| nosotros            | somos       |
| ellos/ellas/ustedes | son         |

## Common Uses

| Spanish                | English                |
| ---------------------- | ---------------------- |
| Soy de México.         | I am from Mexico.      |
| Soy estadounidense.    | I am American.         |
| ¿Eres estudiante?      | Are you a student?     |
| Ella es médica.        | She is a doctor.       |
| Él es un buen médico.  | He is a good doctor.   |
| Es la una.             | It's one o'clock.      |
| Hoy es lunes.          | Today is Monday.       |
| Somos amigos.          | We are friends.        |
| El libro es de María.  | The book is María's.   |
