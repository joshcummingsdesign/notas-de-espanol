# Practicar

"Practicar" means "to practice." It's a regular -ar verb.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | practico    |
| tú                  | practicas   |
| él/ella/usted       | practica    |
| nosotros            | practicamos |
| ellos/ellas/ustedes | practican   |

## Common Uses

| Spanish                        | English                         |
| ------------------------------ | ------------------------------- |
| Practico español todos los días. | I practice Spanish every day. |
| ¿Practicas algún deporte?      | Do you practice any sport?      |
| Ella practica el piano.        | She practices the piano.        |
| Practicamos juntos.            | We practice together.           |
