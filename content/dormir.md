# Dormir

"Dormir" means "to sleep." It's an irregular -ir verb with an o→ue stem change in the present tense.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | duermo      |
| tú                  | duermes     |
| él/ella/usted       | duerme      |
| nosotros            | dormimos    |
| ellos/ellas/ustedes | duermen     |

## Common Uses

| Spanish                       | English                        |
| ----------------------------- | ------------------------------ |
| Duermo ocho horas.            | I sleep eight hours.           |
| ¿Duermes bien?                | Do you sleep well?             |
| El bebé duerme mucho.         | The baby sleeps a lot.         |
| Dormimos tarde los sábados.   | We sleep late on Saturdays.    |
| Ellos duermen en el sofá.     | They sleep on the couch.       |
