# Descansar

"Descansar" means "to rest." It's a regular -ar verb.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | descanso    |
| tú                  | descansas   |
| él/ella/usted       | descansa    |
| nosotros            | descansamos |
| ellos/ellas/ustedes | descansan   |

## Common Uses

| Spanish                       | English                        |
| ----------------------------- | ------------------------------ |
| Necesito descansar.           | I need to rest.                |
| ¿Descansas los fines de semana? | Do you rest on weekends?     |
| Ella descansa después del trabajo. | She rests after work.     |
| Descansamos en el parque.     | We rest in the park.           |
