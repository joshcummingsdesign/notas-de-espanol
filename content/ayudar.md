# Ayudar

"Ayudar" means "to help." It's a regular -ar verb.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | ayudo       |
| tú                  | ayudas      |
| él/ella/usted       | ayuda       |
| nosotros            | ayudamos    |
| ellos/ellas/ustedes | ayudan      |

## Common Uses

| Spanish                    | English                    |
| -------------------------- | -------------------------- |
| ¿Puedo ayudarte?           | Can I help you?            |
| ¿Me puedes ayudar?         | Can you help me?           |
| Necesito ayuda.            | I need help.               |
| Ella siempre me ayuda.     | She always helps me.       |

## Ayudar + A + Infinitive

To say "help to do something," use **ayudar + a + infinitive**:

| Spanish                          | English                        |
| -------------------------------- | ------------------------------ |
| Te ayudo a estudiar.             | I'll help you study.           |
| ¿Puedes ayudarme a cocinar?      | Can you help me cook?          |
| Ella me ayuda a aprender español. | She helps me learn Spanish.   |
