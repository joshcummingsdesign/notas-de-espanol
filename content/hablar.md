# Hablar

"Hablar" means "to speak" or "to talk." It's a regular -ar verb.

## Present Tense (Presente)

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | hablo       |
| tú                  | hablas      |
| él/ella/usted       | habla       |
| nosotros            | hablamos    |
| ellos/ellas/ustedes | hablan      |

## Common Uses

### To Speak

| Spanish                  | English                     |
| ------------------------ | --------------------------- |
| Hablo español.           | I speak Spanish.            |
| ¿Hablas inglés?          | Do you speak English?       |
| Ella habla tres idiomas. | She speaks three languages. |

### Hablar + Con

Use "hablar con" to express talking with someone:

| Spanish                      | English                      |
| ---------------------------- | ---------------------------- |
| Hablo con mi madre.          | I talk with my mother.       |
| ¿Puedo hablar con usted?     | Can I speak with you?        |
| Necesito hablar contigo.     | I need to talk with you.     |
| Estaba hablando con el jefe. | I was talking with the boss. |
