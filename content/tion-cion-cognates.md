# TION/CION Cognates

English words ending in **-tion** almost always become **-ción** in Spanish. This is one of the most reliable patterns for expanding your vocabulary quickly.

## How They Work

Replace **-tion** with **-ción**. The Spanish word will have an accent on the "o" and stress falls on that syllable.

| English       | Spanish       | Pronunciation           |
| ------------- | ------------- | ----------------------- |
| creation      | creación      | creh-ah-see-OHN         |
| invitation    | invitación    | een-bee-tah-see-OHN     |
| documentation | documentación | doh-coo-men-tah-see-OHN |
| information   | información   | een-for-mah-see-OHN     |
| verification  | verificación  | beh-ree-fee-cah-see-OHN |
| nation        | nación        | nah-see-OHN             |
| action        | acción        | ahk-see-OHN             |
| situation     | situación     | see-too-ah-see-OHN      |
| reservation   | reservación   | reh-sehr-bah-see-OHN    |
| education     | educación     | eh-doo-cah-see-OHN      |
| communication | comunicación  | coh-moo-nee-cah-see-OHN |

## Gender: Always Feminine

All **-ción** words are feminine. Always use **la** (not el):

- **la información** - the information
- **la situación** - the situation
- **la nación** - the nation
- **la creación** - the creation

## Creating Verbs: -ción to -r

Replace **-ción** with **-r** to create the infinitive verb form:

| Noun          | Verb       | English        |
| ------------- | ---------- | -------------- |
| creación      | crear      | to create      |
| invitación    | invitar    | to invite      |
| documentación | documentar | to document    |
| información   | informar   | to inform      |
| verificación  | verificar  | to verify      |
| comunicación  | comunicar  | to communicate |
| reservación   | reservar   | to reserve     |
| educación     | educar     | to educate     |

## Watch Out: Spelling Differences

Some words have additional spelling changes beyond -tion to -ción:

| English        | Spanish       | Note               |
| -------------- | ------------- | ------------------ |
| translation    | traducción    | different root     |
| speculation    | especulación  | extra "e" at start |
| authorization  | autorización  | no "h" in Spanish  |
| immigration    | inmigración   | one "m" in Spanish |
| recommendation | recomendación | one "m" in Spanish |
| description    | descripción   | "cc" replaces "ct" |
| introduction   | introducción  | "cc" replaces "ct" |
