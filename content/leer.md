# Leer

"Leer" means "to read." It's an -er verb with spelling changes in some tenses.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | leo         |
| tú                  | lees        |
| él/ella/usted       | lee         |
| nosotros            | leemos      |
| ellos/ellas/ustedes | leen        |

## Common Uses

| Spanish                | English                  |
| ---------------------- | ------------------------ |
| Leo un libro.          | I'm reading a book.      |
| ¿Qué lees?             | What are you reading?    |
| Ella lee el periódico. | She reads the newspaper. |
| Leemos mucho.          | We read a lot.           |
