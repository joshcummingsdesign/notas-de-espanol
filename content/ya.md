# Ya

"Ya" means "already," "now," or "yet" depending on context.

## Examples

| Spanish                  | English                      |
| ------------------------ | ---------------------------- |
| Ya lo sé.                | I already know.              |
| Ya voy.                  | I'm coming (right now).      |
| ¿Ya comiste?             | Did you eat already?         |
| Ya no quiero.            | I don't want to anymore.     |
| Ya es tarde.             | It's already late.           |
| Ya fui.                  | I already went.              |
