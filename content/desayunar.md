# Desayunar

"Desayunar" means "to have breakfast." It's a regular -ar verb.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | desayuno    |
| tú                  | desayunas   |
| él/ella/usted       | desayuna    |
| nosotros            | desayunamos |
| ellos/ellas/ustedes | desayunan   |

## Common Uses

| Spanish                          | English                            |
| -------------------------------- | ---------------------------------- |
| Desayuno a las siete.            | I have breakfast at seven.         |
| ¿Qué desayunas?                  | What do you have for breakfast?    |
| Ella desayuna huevos con tocino. | She has eggs and bacon for breakfast. |
| Desayunamos juntos los domingos. | We have breakfast together on Sundays. |
