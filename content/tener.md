# Tener

"Tener" means "to have." It's an irregular verb with a "go" form in the first person singular and a stem change (e → ie).

## Present Tense (Presente)

Irregular yo form: tengo. Stem change: e → ie in all forms except nosotros.

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | tengo       |
| tú                  | tienes      |
| él/ella/usted       | tiene       |
| nosotros            | tenemos     |
| ellos/ellas/ustedes | tienen      |

## Common Uses

| Spanish                | English                |
| ---------------------- | ---------------------- |
| Tengo un carro.        | I have a car.          |
| ¿Tienes hermanos?      | Do you have siblings?  |
| Ella tiene dos gatos.  | She has two cats.      |

## Tener que + Infinitive

Use "tener que" to express obligation (to have to):

| Spanish                    | English                    |
| -------------------------- | -------------------------- |
| Tengo que trabajar.        | I have to work.            |
| Tienes que estudiar.       | You have to study.         |
| Tenemos que salir ahora.   | We have to leave now.      |
