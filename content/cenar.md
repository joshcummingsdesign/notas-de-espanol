# Cenar

"Cenar" means "to have dinner." It's a regular -ar verb.

## Present Indicative

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | ceno        |
| tú                  | cenas       |
| él/ella/usted       | cena        |
| nosotros            | cenamos     |
| ellos/ellas/ustedes | cenan       |

## Common Uses

| Spanish                        | English                          |
| ------------------------------ | -------------------------------- |
| Ceno a las ocho.               | I have dinner at eight.          |
| ¿A qué hora cenas?             | What time do you have dinner?    |
| Cenamos en un restaurante.     | We have dinner at a restaurant.  |
| Ellos cenan tarde.             | They have dinner late.           |
