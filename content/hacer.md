# Hacer

"Hacer" means "to do" or "to make." It's an irregular verb with a "go" form in the first person singular.

## Present Tense (Presente)

Irregular yo form: hago

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | hago        |
| tú                  | haces       |
| él/ella/usted       | hace        |
| nosotros            | hacemos     |
| ellos/ellas/ustedes | hacen       |

## Common Uses

### To Do

| Spanish           | English                  |
| ----------------- | ------------------------ |
| ¿Qué haces?       | What are you doing?      |
| Hago mi tarea.    | I'm doing my homework.   |
| ¿Qué hacemos hoy? | What are we doing today? |

### To Make

| Spanish              | English              |
| -------------------- | -------------------- |
| Hago la cena.        | I'm making dinner.   |
| Ella hace un pastel. | She's making a cake. |
| Hacemos planes.      | We're making plans.  |

### Weather Expressions

"Hacer" is used in many weather expressions:

| Spanish           | English              |
| ----------------- | -------------------- |
| Hace calor.       | It's hot.            |
| Hace frío.        | It's cold.           |
| Hace sol.         | It's sunny.          |
| Hace viento.      | It's windy.          |
| Hace buen tiempo. | The weather is nice. |

### Hacer + Time Expression (Ago)

| Spanish              | English        |
| -------------------- | -------------- |
| Hace dos años.       | Two years ago. |
| Hace una hora.       | An hour ago.   |
| ¿Hace cuánto tiempo? | How long ago?  |
