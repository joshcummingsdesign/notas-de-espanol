# Comer

"Comer" means "to eat." It's a regular -er verb.

## Present Tense (Presente)

| Pronoun             | Conjugation |
| ------------------- | ----------- |
| yo                  | como        |
| tú                  | comes       |
| él/ella/usted       | come        |
| nosotros            | comemos     |
| ellos/ellas/ustedes | comen       |

## Common Uses

| Spanish                | English                  |
| ---------------------- | ------------------------ |
| Como el desayuno.      | I eat breakfast.         |
| ¿Qué comes?            | What are you eating?     |
| Ella come mucha fruta. | She eats a lot of fruit. |
| Comemos a las seis.    | We eat at six.           |
